## Project Name

2048 GAME USING DESIGN PATTERNS

## Project Description

This project is a re-implementation of the classic 2048 game. Used design patterns:

- State pattern
- Decorator pattern
- Template pattern
- Observer pattern
- Command pattern
- Strategy pattern
- Prototype pattern
